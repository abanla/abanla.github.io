标签插件（Tag）

标签插件帮助开发者在文章中快速插入内容。

概要

标签函数会传入两个参数：args 和 content，前者代表开发者在使用标签插件时传入的参数，而后者则是标签插件所覆盖的内容。

从 Hexo 3 开始，因为新增了异步渲染功能，而改用 Nunjucks 作为渲染引擎，其行为可能会与过去使用的 Swig 有些许差异。

移除标签插件

使用 unregister() 来用自定义函数替换现有的 标签插件。

示例

选项

ends

使用结束标签，此选项默认为 false。

async

开启异步模式，此选项默认为 false。
